# ADVAITALOKA GitHub Pages Lite Bundle

This is the lighter upload package for GitHub Pages.

Use this if the larger browser upload fails during commit.

## What it includes

- the full public website
- English and Hindi pages
- core images and styling
- linked markdown documents used by the site
- meeting-ready readme folders

## What it excludes

- large `.pptx` and `.xlsx` files that often cause browser upload failures

## Publish steps

1. Open your `advaitaloka-site` repository on GitHub.
2. If the failed upload left partial files, remove them or start with a fresh empty repo.
3. Click `uploading an existing file`.
4. Upload the contents of `advaitaloka-github-pages-lite-upload.zip`.
5. Commit changes.
6. Open `Settings -> Pages`.
7. Set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
8. Save.
