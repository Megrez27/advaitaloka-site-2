# सहजीवनम् पुरम् – पिच तैयारी नोट्स (V1)

## उद्देश्य

यह नोट उन प्रमुख प्रश्नों का उत्तर देता है जो सरकार, निवेशकों, SPV भागीदारों, या संस्थागत सहयोगियों द्वारा पिच के समय सबसे पहले उठाए जा सकते हैं।

## 1. वित्तीय मॉडल को अधिक ठोस बनाना होगा

- INR 5,200 करोड़ की headline संख्या अकेले पर्याप्त नहीं है।
- phased capex, funding mix, revenue timing, और cash-flow logic स्पष्ट होने चाहिए।
- Phase 0 ask को total project envelope से अलग दिखाना आवश्यक है।

## 2. "Sovereign Network State" शब्दावली जोखिमपूर्ण है

यह शब्द visionary है, परंतु सरकारी स्वीकृति-संदर्भ में कानूनी और राजनीतिक आशंका उत्पन्न कर सकता है।

### सरकारी उपयोग हेतु बेहतर शब्द

- AI-enabled Accessibility Township
- Inclusive Innovation Zone
- SPV-led Demonstration District

## 3. दिव्यांगजन का परिमाण स्पष्ट होना चाहिए

आधिकारिक legacy baseline:

- उत्तर प्रदेश: 41,57,514 दिव्यांगजन
- कुल आबादी का लगभग 2.08%

यह संख्या आवश्यकता के पैमाने को दिखाने के लिए पर्याप्त है, पर इसे current live estimate की तरह नहीं प्रस्तुत करना चाहिए।
