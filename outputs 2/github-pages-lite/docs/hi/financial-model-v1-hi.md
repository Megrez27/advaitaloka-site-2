## 4. Scenario Cases

वर्कबुक में अब तीन operating cases शामिल हैं:

- `Conservative`:
  slower workforce ramp, softer mentorship realization, delayed IP commercialization, और `10%` Phase 0 cost uplift
- `Base`:
  मुख्य dashboard और narrative में उपयोग होने वाला midpoint case
- `Upside`:
  stronger workforce utilization, better mentorship conversion, faster IP commercialization, और बिना अतिरिक्त Phase 0 inflation uplift

ये cases safe external conversations के लिए बनाए गए हैं। इन्हें committed revenue forecast की तरह प्रस्तुत नहीं किया जाना चाहिए।

## 5. Sensitivity Views

वर्कबुक में दो quick sensitivity grids शामिल हैं:

1. `Year 7 Revenue Sensitivity`
   Digital workforce realization बनाम IP licensing realization
2. `Year 7 Revenue / Phase 0 Ask`
   Digital workforce realization बनाम Phase 0 budget inflation

इनका उपयोग उन सामान्य प्रश्नों के उत्तर में किया जा सकता है:

- अगर workforce execution धीमा रहा तो क्या होगा,
- अगर IP monetization देर से शुरू हुई तो क्या होगा,
- और revenue story कितनी हद तक Phase 0 cost discipline पर निर्भर है।

## 9. सुरक्षित pitching language

- bankability claim अभी न करें।
- guaranteed returns claim न करें।
- scenario outputs को contracted revenue की तरह न रखें।
- workbook को Phase 0 authorization और deeper diligence justify करने वाले concept-stage tool की तरह प्रस्तुत करें।
