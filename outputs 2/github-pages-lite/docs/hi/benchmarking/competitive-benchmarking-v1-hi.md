# सहजीवनम् पुरम् – तुलनात्मक बेंचमार्किंग नोट (V1)

## उद्देश्य

यह नोट परियोजना की विश्वसनीयता बढ़ाने हेतु relevant precedent fragments को व्यवस्थित करता है।

## 1. Auroville

### क्या दिखाता है

- experimental township imagination
- values-led settlement vision
- long-horizon place identity

### सीमाएँ

- accessibility-first infrastructure नहीं
- special-zone governance नहीं
- digital civic operating system नहीं

## 2. GIFT City

### क्या दिखाता है

- state-backed flagship positioning
- concentrated infrastructure strategy
- special-zone governance logic
- institutional legibility

### सीमाएँ

- accessibility-centered planning नहीं
- disability inclusion model नहीं

## 3. Enabling Village

### क्या दिखाता है

- integrated disability inclusion
- accessible public space
- community services, training, and employment programming

### सीमाएँ

- township-scale model नहीं
- state-zone finance structure नहीं

## 4. समेकित निष्कर्ष

सहजीवनम् पुरम् को सबसे प्रभावी ढंग से इस प्रकार समझाया जा सकता है:

- Auroville जैसी township imagination
- GIFT City जैसी governance seriousness
- Enabling Village जैसी accessibility integration
