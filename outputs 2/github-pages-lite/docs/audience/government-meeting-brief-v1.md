## Core Message

The right first decision is:

1. approve structured Phase 0 review,
2. support SPV and pathway design,
3. validate land, demand, and accessibility planning,
4. return after Phase 0 with sharper costing and institutional structuring.

## What To Emphasize In The Room

- approval-safe framing:
  AI-enabled Accessibility Township, Inclusive Innovation Zone, SPV-led Demonstration District
- Uttar Pradesh relevance:
  disability baseline, elder inclusion, employment, and standards leadership
- bounded first ask:
  `INR 22 crore` for Phase 0, not `INR 5,200 crore` immediate sanction
- governance discipline:
  privacy, consent, audits, stage gates, accessibility oversight

## Questions You Should Be Ready For

1. Why Uttar Pradesh first?
2. Why now?
3. Why is the first ask `INR 22 crore`?
4. What exactly does Phase 0 deliver?
5. Why should the state support an SPV structure?

## Recommended Close

Ask for a structured review pathway, nodal ownership, and Phase 0 authorization rather than immediate full capital commitment.
